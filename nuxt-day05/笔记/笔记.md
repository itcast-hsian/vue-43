# 首页的搜索框

1.去掉空白值的时候去掉下拉框

```js
queryDestSearch(value, cb) {
            // // 如果value值空的，就不需要请求
            if(!value){
                cb([]);
                return;
            }

```

> 重点是这个: `cb([]);`



2.如果输入框为空，去除默认选中的第一城市

```
queryDepartSearch(value, cb) {  
            
            // 如果value值空的，就不需要请求
            if(!value){
                // 禁止输入框的值是空的时候显示下拉框
                cb([]);
                // 如果输入框的值是空的话把之前的城市列表删除掉
                this.departCities = [];
                return;
            }
```

> 重点是这个: `this.departCities = [];`
